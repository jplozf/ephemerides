<p class="Presentation">
<span style="text-decoration: underline;">Note&nbsp;:</span><br>
* &Delta;T values from year 1620 onward are taken from a table based on F.R. Stephenson &amp; L.V. Morrison (1984) work studying historical records. In modern times values are essentially based on VLB (Very Long Baseline) observations of radio sources such as quasars. Values since 1973 are given by the following formula <tt>&Delta;T = TAI - UT1 + 32.184 sec = (TAI - UTC) - (UT1 - UTC) + 32.184 sec</tt>.<br>
Values prior to year 1620 are computed using the following formulas&nbsp;:<br>
<tt>&Delta;T (sec) = 2177 + 497 * u + 44.1 * u&sup2;</tt> (<i>u</i> = centuries from 2000) for years between -391&nbsp;BCE and +948&nbsp;CE<br>
<tt>&Delta;T (sec) = 102 + 102 * u + 25.3 * u&sup2;</tt> (<i>u</i> = centuries from 2000) for years between +948&nbsp;CE and +1600&nbsp;CE<br>
from Jean Chapront, Michelle Chapront-Touz&eacute; &amp; G. Francou (1997), using a Moon&rsquo;s secular acceleration value (<i>n'</i>) of -25.7376&quot;/cy&sup2; (where cy is given in centuries).<br>
Future values of &Delta;T can be estimated at best.
</p>
